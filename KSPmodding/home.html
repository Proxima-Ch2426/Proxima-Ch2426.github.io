Unity vs Blender

Even though the Blender mu addon is a modern alternative compared to the aging Unity, I still personally use Unity, not because it's better, but because I am too lazy to change my workflow lol. If you are looking for a tutorial on how to make KSP mods using the Blender workflow then this guide is not for you, but there are plenty of useful guides for the Blender workflow. However, I do plan to move over some of my workflow to Blender especially for small parts, just to try to get used to the Blender workflow in the event that Unity spontaneously combusts for some reason.
