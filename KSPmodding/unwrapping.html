Unwrapping

Helpful Blender addons

  • Texel Density Check - Check UV pixel density
  • TexTools - UV utilites

It's recommended to use a texel density of 200px/m. Larger surfaces can have lesser density, and smaller surfaces can go higher density. I personally have been using 150px/m because most of the parts i've made so far are rather large.

Try to fill up your UV map as much as possible and minimise wasted space. Small parts can even use the same texture to save on memory.

Try not to use TexTool's Rectify function on steep conical surfaces as it might cause stretching between the top and bottom of the mesh.
